# LLM Setup Instructions

Paste the following prompt into your AI coding assistant (Claude Code, Cursor, Copilot, etc.) to install heic2jpg:

> Clone https://github.com/brennancheung/heic2jpg.git, then run `chmod +x install.sh uninstall.sh && ./install.sh`. If `swiftc` is not available, run `xcode-select --install` first. After installing, verify the daemon is running with `launchctl list | grep heic2jpg`.
